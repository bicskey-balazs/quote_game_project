Nincs angol TTS hang telepítve ezen a gépen!
A felolvasás magyarul fog történni, vagy nem lesz természetes az angol kiejtés.

Elérhető felolvasó telepítése:
Windows-on:
  • Gépház → Idő és nyelv → Beszéd
  • „Hangok hozzáadása”
  • Válassz „English (United States)” vagy más angol hangot
  • Böngésző újraindítása
    • Mac-en:

    System Settings → Accessibility → Spoken Content → System Voice

    Ajánlott böngészők: Microsoft Edge, Firefox, Google Chrome (néha működik)
